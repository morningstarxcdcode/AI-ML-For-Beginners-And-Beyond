# Machine Learning Algorithms Cheat Sheet

## 1. **Linear Regression**

- **Type**: Supervised
- **Purpose**: Predict continuous target variable.
- **Example**: House price prediction based on square footage.

## 2. **Logistic Regression**

- **Type**: Supervised
- **Purpose**: Binary classification.
- **Example**: Predicting if an email is spam or not.

## 3. **K-Nearest Neighbors (KNN)**

- **Type**: Supervised
- **Purpose**: Classification or regression based on nearest neighbors.
- **Example**: Classifying plants based on their features.

## 4. **Support Vector Machines (SVM)**

- **Type**: Supervised
- **Purpose**: Classification, finds the optimal hyperplane to separate data.
- **Example**: Classifying images as cats or dogs.

## 5. **Decision Trees**

- **Type**: Supervised
- **Purpose**: Classification or regression based on decision rules.
- **Example**: Predicting customer churn.

## 6. **Random Forest**

- **Type**: Supervised
- **Purpose**: Ensemble learning method using multiple decision trees.
- **Example**: Predicting credit card fraud.

## 7. **K-Means Clustering**

- **Type**: Unsupervised
- **Purpose**: Group data into k clusters based on similarities.
- **Example**: Customer segmentation in marketing.

## 8. **Principal Component Analysis (PCA)**

- **Type**: Unsupervised
- **Purpose**: Dimensionality reduction.
- **Example**: Reducing the number of features for large datasets.

## 9. **Naive Bayes**

- **Type**: Supervised
- **Purpose**: Classification based on probability.
- **Example**: Sentiment analysis for movie reviews.

## 10. **Gradient Boosting**

- **Type**: Supervised
- **Purpose**: Ensemble technique that builds models sequentially.
- **Example**: Predicting customer satisfaction.
