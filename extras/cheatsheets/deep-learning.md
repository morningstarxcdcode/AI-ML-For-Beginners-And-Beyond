# Deep Learning Algorithms Cheat Sheet

## 1. **Artificial Neural Networks (ANN)**

- **Purpose**: Mimics the human brain, learning complex patterns.
- **Use Cases**: Image recognition, speech recognition.
- **Common Architecture**: Input layer, hidden layers, output layer.

## 2. **Convolutional Neural Networks (CNN)**

- **Purpose**: Specialized for image processing.
- **Use Cases**: Image classification, object detection.
- **Key Components**: Convolutional layers, pooling layers, fully connected layers.

## 3. **Recurrent Neural Networks (RNN)**

- **Purpose**: Suitable for sequential data like time-series.
- **Use Cases**: Text generation, speech recognition, language translation.
- **Key Variants**: LSTM, GRU.

## 4. **Generative Adversarial Networks (GANs)**

- **Purpose**: Generate new data samples that resemble the original data.
- **Use Cases**: Image generation, text-to-image synthesis.
- **Architecture**: Generator and discriminator networks.

## 5. **Autoencoders**

- **Purpose**: Data compression and noise reduction.
- **Use Cases**: Image denoising, anomaly detection.
- **Key Components**: Encoder, bottleneck, decoder.

## 6. **Long Short-Term Memory (LSTM)**

- **Purpose**: A type of RNN designed to avoid vanishing gradient problem.
- **Use Cases**: Speech-to-text, stock price prediction.

## 7. **Transformer**

- **Purpose**: Handle sequence-to-sequence tasks without using RNNs.
- **Use Cases**: Machine translation, text summarization, BERT, GPT models.
