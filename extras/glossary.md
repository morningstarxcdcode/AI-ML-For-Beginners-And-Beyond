# 📘 AI/ML Glossary

**Model**: A program trained to make predictions using data.

**Training**: Teaching the model using known data.

**Supervised Learning**: Training with labeled data (we tell the model what's correct).

**Unsupervised Learning**: No labels — the model finds patterns on its own.

**Neural Network**: A computer model inspired by the human brain.

**Overfitting**: Model learns the training data too well but fails on new data.

**Bias**: A consistent error in predictions (often due to bad data or assumptions).

...and so on!
